# Conversa com IA
Uma aplicação que recebe uma pergunta em audio e responde à pergunta também em audio.  
Essa aplicação foi desenvolvida no Google Colab e utilizou as seguintes bibliotecas:
* whisper: para transformar áudio em texto
* gpt4all: para gerar a resposta
* gTTS: para converter a resposta em áudio

O endereço da aplicação no Google Colab é https://colab.research.google.com/drive/1GR1RX2rYY__9qJdCkbp84FuMs0L128mc#scrollTo=X2W9qrmBvMsn.
